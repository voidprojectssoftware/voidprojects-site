# voidprojects-site docs

Documentation is organized with [Diátaxis](https://diataxis.fr/): within each topic, content is
split by the reader's need into **explanation** (understand it), **how-to guides** (do a task),
and **reference** (look something up). Pick the topic, then the mode that matches what you need.
